# 安装与使用

## 1. 加载扩展

1. 打开 Edge 或 Chrome
2. 进入扩展管理页
3. 开启“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择 `apps/extension`

如果你后续修改了扩展代码，需要回到扩展页点一次“重新加载”。
平时正常使用时，不需要每次开机都重新加载。

## 2. 记录一条聊天

1. 打开你想保存的 ChatGPT 会话
2. 刷新当前 ChatGPT 页面一次
3. 打开扩展弹窗
4. 点击 `读取当前聊天信息`
5. 按需修改 `归档名称（可自定义）`
6. 点击 `记录当前聊天`

同一条会话再次记录时，会自动按会话 ID 追加新消息，不会新建重复归档。

## 3. 查看归档

1. 打开扩展弹窗
2. 点击 `打开归档查看器`

在归档查看器里你可以：

- 搜索聊天
- 查看消息内容
- 修改归档名称
- 导出单条 Markdown
- 导出整组 Markdown
- 删除单条归档
- 删除整组归档

## 4. 数据位置

当前版本把数据保存在浏览器扩展本地存储中，不需要本地服务。

这意味着：

- 换电脑后不会自动同步
- 删除扩展或清理扩展数据后，归档可能消失
- 建议定期导出重要对话

## 5. 分享给别人使用

直接把 `apps/extension` 文件夹打包发给别人即可。

别人解压后，也按照“加载已解压的扩展程序”的方式安装。
